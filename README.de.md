KI-gestuetztes Competitive-Intelligence-Toolkit fuer **Product Manager / PMM**.

## Schnellstart

```bash
pip install -r requirements.txt
python main.py --competitors "WettbewerberA,WettbewerberB" --market "SaaS CRM"
```

- Berichtsausgabe: `outputs/agent/`
- API-Schluessel in `.env` setzen (z. B. `ZHIPU_API_KEY_1`)
