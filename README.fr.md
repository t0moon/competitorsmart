Boite a outils d'intelligence concurrentielle assistee par IA pour **Product Managers / PMM**.

## Demarrage rapide

```bash
pip install -r requirements.txt
python main.py --competitors "ConcurrentA,ConcurrentB" --market "SaaS CRM"
```

- Sortie des rapports: `outputs/agent/`
- Configurez votre cle dans `.env` (par exemple `ZHIPU_API_KEY_1`)
