**프로덕트 매니저 / PMM** 를 위한 AI 경쟁 정보 분석 도구입니다.

## 빠른 시작

```bash
pip install -r requirements.txt
python main.py --competitors "경쟁사A,경쟁사B" --market "SaaS CRM"
```

- 리포트 출력 경로: `outputs/agent/`
- `.env` 에 API 키(예: `ZHIPU_API_KEY_1`)를 설정하세요
