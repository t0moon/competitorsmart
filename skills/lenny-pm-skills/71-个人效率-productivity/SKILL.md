# 个人效率

## 何时使用此技能

当你需要：
- 管理高强度的工作负荷
- 平衡多个角色和职责
- 建立有效的任务管理系统
- 提升专注力和执行力

## 核心原则

### 1. 时间块管理
通过将时间分块来管理高强度的全职工作和多重角色。

### 2. 一切都写下来
把事情写下来能帮助更好地记忆和管理任务。

### 3. 系统化任务管理
建立系统化的待办事项清单来保持心智清晰。

## 专家洞察

### Gokul Rajaram
> "实际上就是时间块管理和知道……我周六周日各做两小时，这样我就可以做四场会议。"

**核心洞察**：这位专家讨论了具体的"时间块"技术，用于管理高强度的全职工作以及董事会席位和大量投资活动。

**如何应用**：
- 为不同类型的工作设定固定的时间块
- 利用周末时间处理额外的会议和事务
- 明确每个时间块的具体用途

### Maya Prohovnik
> "我把所有事情都写下来，因为我是那种写下来才能记住的人，然后我会疯狂地把事情放到待办清单上。"

**核心洞察**：这位专家讨论了管理高强度领导角色的具体系统，包括任务管理和心智清晰技术。

**如何应用**：
- 养成把一切都写下来的习惯
- 建立并维护详细的待办事项清单
- 通过书写来释放大脑空间

## 常见错误

- 试图记住所有事情而不记录
- 没有明确的时间块划分
- 工作时间与个人时间界限模糊
- 待办清单过于笼统缺乏具体性
- 不愿意利用周末时间处理溢出工作

## 关键战术

| 战术 | 说明 |
|------|------|
| **时间块法** | 为不同任务类型分配固定时间段 |
| **全面记录** | 把所有事情都写下来释放大脑 |
| **待办清单** | 维护系统化的任务追踪系统 |
| **周末补充** | 利用周末处理额外的会议和任务 |
