# 创业构思

## 何时使用此技能

当你需要：
- 寻找创业方向和想法
- 评估创业想法的可行性
- 避免常见的创业陷阱
- 分析"为什么是现在"

## 核心原则

### 1. 走非寻常路
从个人经验出发，寻找独特的机会，避免热门但拥挤的领域。

### 2. 分析"为什么是现在"
思考什么新技术或趋势使得今天可以做昨天做不了的事。

### 3. 多元化信息来源
拓展信息渠道，获取独特洞察。

## 专家洞察

### Dalton Caldwell
> "尝试更多地走非寻常路，无论是从你的个人经验出发..."

**核心洞察**：如何选择初始想法、避免"焦油坑"（看起来诱人但难以逃脱的想法），以及多元化"信息饮食"来发现独特机会。

**如何应用**：
- 避免追逐热门但拥挤的领域
- 从个人独特经验中挖掘机会
- 识别和避开"焦油坑"想法
- 拓展信息来源获取差异化洞察

### Ryan Hoover
> "今天你能做什么昨天做不了的新事情？有很多类别，比如Web3、AI。"

**核心洞察**：寻找新想法的具体框架，包括观察技术变革、消费者行为变化，以及"为什么是现在"的分析。

**如何应用**：
- 分析新技术创造的可能性
- 观察消费者行为的变化趋势
- 回答"为什么是现在"这个关键问题
- 在技术转折点寻找机会

## 常见错误

- 追逐热门领域，进入"焦油坑"
- 想法太大太泛，无法落地执行
- 只看技术可能性，忽视真实需求
- 信息来源单一，缺乏独特洞察

## 关键战术

| 战术 | 说明 |
|------|------|
| **个人经验挖掘** | 从自己遇到的痛点和独特经历出发 |
| **"为什么是现在"分析** | 识别使想法可行的技术或社会变化 |
| **焦油坑检测** | 评估想法是否是看起来好但难以成功的陷阱 |
| **信息多元化** | 拓展信息渠道，获取差异化洞察 |
