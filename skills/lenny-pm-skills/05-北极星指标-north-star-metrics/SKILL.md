# 北极星指标

## 何时使用此技能

当你需要：
- 定义产品或团队的核心成功指标
- 在复杂生态系统中简化决策
- 找到可实验的短期指标来预测长期结果
- 从客户视角定义价值衡量标准

## 核心原则

### 1. 避免用滞后指标做目标
留存率等滞后指标很难在短期内有意义地推动。找到能预测长期结果的短期输入指标。

### 2. 简化复杂决策
清晰的高层北极星可以简化复杂、相互关联的产品生态系统中的决策。

### 3. 从客户视角定义指标
最好的指标从客户视角衡量价值的成功传递，而非内部活动。

### 4. 先定性后定量
北极星指标在作为定性模型帮助团队对齐价值认知时最有效，然后再添加测量。

### 5. 资源围绕北极星组织
管理的本质是定义清晰的结果，并组织资源去实现它。

## 专家洞察

### Jess Lachs
> "留存率是一个糟糕的目标。短期内几乎不可能有意义地推动。最终，你想找到一个可以测量的短期指标，它能驱动长期结果。"

**核心洞察**：避免用留存率等滞后指标作为主要目标；相反，识别并推动能预测它们的短期输入指标。

**如何应用**：
- 识别对实验敏感且与长期目标相关的代理指标
- 专注于团队能在季度或月度时间框架内实际影响的输入

**如何应用**：

## 常见错误

- 用留存率等滞后指标做目标
- 指标太多导致失焦
- 从内部活动而非客户价值定义指标
- 只有定量没有定性模型
- 北极星与日常工作脱节

## 关键战术

| 战术 | 说明 |
|------|------|
| **输入vs输出** | 区分可影响的输入和滞后的输出 |
| **客户视角** | 问「客户会以此为荣吗？」 |
| **零摩擦指标** | 用负面事件的缺失作为成功标志 |
| **三种游戏** | 识别北极星输入的运作模式 |
| **单一仲裁** | 让北极星成为决策的最终仲裁 |
