# 精力管理

## 何时使用此技能

当你需要：
- 保持长期高效而不倦怠
- 优化日程安排以匹配精力水平
- 识别消耗或补充精力的活动
- 更多地工作在"天才区"

## 核心原则

### 1. 管理精力而非时间
高效能不仅仅是时间管理，更重要的是管理你的精神和情感能量，将高能量时段用于最重要的工作。

### 2. 识别能量驱动和消耗
识别哪些活动给你带来能量（能量驱动），哪些活动让你感到疲惫（能量消耗），据此设计工作和角色。

### 3. 聚焦天才区
审视日历，将时间分类，然后将精力集中在让你处于"天才区"的工作上，委派或消除消耗能量的任务。

**核心洞察**：提供了关于管理精神和情感能量而非仅仅管理时间以保持高绩效的具体战术建议。

**如何应用**：
- 根据精力水平安排工作时间
- 将高能量时段用于最重要的任务

**核心洞察**：介绍了一个识别"能量驱动"和"能量消耗"的具体框架，用于管理个人生产力、防止倦怠和设计角色。

**如何应用**：
- 记录给你带来能量的活动
- 识别消耗你能量的任务
- 据此调整工作安排

**核心洞察**：提供了一个审视日历以找到"天才区"工作并委派消耗能量任务的具体框架。

**核心洞察**：基于追踪每次互动或任务后什么"给你能量"与"消耗能量"的职业方向框架。

**如何应用**：
- 在每次会议或任务后记录能量状态
- 追踪模式以指导职业决策

## 常见错误

- 只管理时间而忽视精力水平
- 不识别哪些活动给你带来或消耗能量
- 将高能量时段浪费在低价值任务上
- 长期忽视能量消耗，导致倦怠

## 关键战术

| 战术 | 说明 |
|------|------|
| **能量追踪** | 记录每次活动后的能量状态，识别模式 |
| **日历审计** | 分类日历活动，识别"天才区"工作 |
| **能量匹配** | 将高能量时段安排给最重要的工作 |
| **委派消耗** | 委派或消除持续消耗能量的任务 |
| **物理工具** | 使用减压玩具等物理工具管理神经负荷 |
