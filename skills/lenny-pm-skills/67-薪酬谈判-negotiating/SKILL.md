# 薪酬谈判

## 何时使用此技能

当你需要：
- 谈判新工作的薪酬待遇
- 争取更好的股权方案
- 协商工作条件和资源
- 将全职工作转为合同制

## 核心原则

### 1. 先谈资源再谈薪酬
在讨论个人薪酬之前，先协商你成功所需的资源、预算和组织承诺。

### 2. 薪酬是其他需求的替代品
薪酬往往是尊重或学习等其他需求的替代品，优化体验可能比最大化薪资更有价值。

### 3. 灵活的工作形式
从全职转为合同制可以提供职业转型所需的灵活性，同时保持收入。

## 专家洞察

### Phyl Terry
> "我希望你去和招聘经理谈谈……我希望你说：'这很棒。我想谈谈薪酬，但在此之前，我想考虑一些能让我在这个角色中成功的条件。我认为这里有大约1000万美元的技术债务，对吗？……'"

**核心洞察**：在讨论个人薪酬之前，先协商你成功所需的资源、预算和组织承诺。

**如何应用**：
- 在面试过程中识别关键障碍（如技术债务、团队培训、人员配置）
- 将解决这些障碍的具体预算或权限作为offer谈判的一部分
- 将这些请求框定为实现约定OKR所必需的条件

### Bob Moesta
> "如果我给人们更好的体验，我实际上可以支付更少的薪酬……我付给他们相当不错的薪水，但我知道他们可以在别处赚更多钱。最终我希望他们被吸引离开。所以如果他们需要更多钱，他们应该去别处。但如果他们是来学习的，那就是我想要的……"

**核心洞察**：薪酬往往是尊重或学习等其他需求的替代品；优化体验可能比最大化薪资更有价值。

**如何应用**：
- 识别候选人是优先考虑"功能"（薪资/头衔）还是"体验"（学习/自主权）
- 对权衡保持透明，比如用较低的基本薪资换取高成长的学习机会或导师指导

### Paul Millerd
> "很多人做的一个重大举措是把他们当前的工作转为合同工作，很多雇主对此比人们想象的更开放……你可以说：'嘿，我想每周工作三天。我想成为合同工。这是我想做的具体内容。你对此开放吗？'"

**核心洞察**：在同一家公司从全职转为合同制可以提供职业转型所需的灵活性，同时保持收入。

**如何应用**：
- 向当前雇主提议具体的每周三天合同制结构
- 强调合同安排对雇主的好处（更容易终止合作降低风险）

## 常见错误

- 一上来就谈薪资数字
- 忽视非薪酬条件的谈判
- 不了解自己真正的优先级（薪资vs学习vs灵活性）
- 假设雇主不会接受非传统安排
- 没有为成功争取必要的资源

## 关键战术

| 战术 | 说明 |
|------|------|
| **资源先行** | 先谈成功所需的预算和权限，再谈薪酬 |
| **识别真正需求** | 区分"功能需求"（薪资头衔）和"体验需求"（学习成长） |
| **合同制转换** | 提议将全职转为灵活的合同制安排 |
| **OKR挂钩** | 将资源请求与实现目标的必要条件相关联 |
