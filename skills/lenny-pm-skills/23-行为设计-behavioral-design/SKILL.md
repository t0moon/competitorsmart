# 行为设计

## 何时使用此技能

当你需要：
- 运用心理学原理设计产品功能
- 提升用户留存和参与度
- 设计习惯养成机制
- 理解用户行为动机

## 核心原则

### 1. 利用心理偏见
损失厌恶、现状偏好、不确定性厌恶等心理偏见可以被用于产品设计。

### 2. 创造暂停时刻
使用触觉反馈和动画创造"暂停时刻"来强化用户行为。

### 3. 弯曲而非打破
设计应该"弯曲"用户习惯而非强行"打破"。

## 专家洞察

### Jackson Shuttleworth
> "一旦你达到七天，损失厌恶就会生效，你就会留住用户。"

**核心洞察**：深层心理学原理如损失厌恶、"弯曲而非打破"哲学，以及使用触觉反馈/动画创造"暂停时刻"来庆祝，这些都超越了标准的留存策略。

**如何应用**：
- 设计7天以上的连续使用机制触发损失厌恶
- 使用动画和触觉反馈创造仪式感
- 在关键时刻设计"暂停"让用户感受成就

### Kristen Berman
> "行为科学和行为设计基本上是利用这些心理学洞察，将其应用于解决现实世界的问题。"

**核心洞察**：将行为经济学（如现时偏好、现状效应、不确定性厌恶等偏见）应用于产品功能，以驱动特定用户行为。

**如何应用**：
- 利用现时偏好设计即时奖励
- 利用现状效应设计默认选项
- 减少不确定性降低用户决策成本

## 常见错误

- 将行为设计等同于操纵用户
- 忽视长期用户价值只追求短期指标
- 过度使用心理技巧导致用户反感
- 不理解底层心理原理盲目套用技巧

## 关键战术

| 战术 | 说明 |
|------|------|
| **损失厌恶设计** | 设计连续打卡等机制，让用户不愿失去已有进度 |
| **默认选项优化** | 利用现状偏好，设计对用户有利的默认选项 |
| **即时反馈** | 使用动画、声音、触觉提供即时正向反馈 |
| **降低摩擦** | 减少决策点和不确定性，降低行动成本 |
