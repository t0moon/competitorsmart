# 媒体关系

## 何时使用此技能

当你需要：
- 与记者和媒体建立长期关系
- 策划新闻发布和媒体报道
- 准备媒体采访和新闻稿件
- 理解如何有效地向记者推销故事

## 核心原则

### 1. 独家策略优先
提供独家新闻比群发禁令邮件更能获得高质量报道。

### 2. 系统化推销流程
成功的媒体关系遵循准备、识别、推销的系统化流程。

### 3. 建立长期关系
媒体关系是长期投资，重点在于建立可持续的互惠关系。

## 专家洞察

### Emilie Gerber
> "我会错开时间，因为我们提供的是独家新闻，我强烈建议提供独家新闻。"

**核心洞察**：管理记者关系、提供独家新闻、驾驭媒体生态是一套超越通用"发布营销"或"定位"的深度战术建议。

**如何应用**：
- 错开推销时间，一次只联系一个记者
- 将故事作为独家新闻提供，增加被采纳的可能性
- 建立记者关系数据库，追踪偏好和历史互动
- 在提供独家时明确传达独家的条件

### Jason Feifer
> "第一步是准备...第二步是找出向谁推销...第三步是实际的推销。"

**核心洞察**：识别、推销和管理与记者及自由撰稿人关系的综合框架，是一套独特于通用营销的专业技能。

**如何应用**：
- **准备阶段**：
  - 明确你的故事角度和新闻价值
  - 准备好所有支持材料（数据、引用、图片）
  - 了解当前媒体趋势和时机

- **识别阶段**：
  - 研究哪些记者和媒体覆盖你的领域
  - 了解每个记者的偏好、风格和历史报道
  - 建立分层的目标媒体列表

- **推销阶段**：
  - 个性化每封推销邮件
  - 在主题行中传达核心价值
  - 保持简洁，快速说明新闻价值
  - 提供明确的后续步骤

## 常见错误

- 同时向多个记者发送相同的推销邮件
- 没有研究记者的报道领域和风格
- 推销内容过于自我宣传而缺乏新闻价值
- 没有准备好支持材料就仓促联系媒体
- 只在需要报道时才联系记者，没有建立关系

## 关键战术

| 战术 | 说明 |
|------|------|
| **独家策略** | 一次只向一个记者提供独家新闻 |
| **三步流程** | 遵循准备-识别-推销的系统化流程 |
| **记者研究** | 深入了解目标记者的偏好和历史报道 |
| **关系维护** | 建立长期关系而非仅在需要时联系 |
| **时机把握** | 了解媒体周期和最佳推销时机 |
