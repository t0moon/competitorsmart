## 使用方法

### Claude Code / Claude Desktop
将此仓库克隆到 `~/.claude/skills/` 目录下即可自动加载。

```bash
git clone https://github.com/YOUR_USERNAME/lenny-pm-skills.git ~/.claude/skills/lenny-pm-skills
```

### Obsidian
可直接作为 Obsidian vault 使用，支持 wikilink 互联。

## 技能结构

每个技能文件包含：
- **何时使用** - 触发场景
- **核心原则** - 5个核心原则
- **专家洞察** - 来自产品领袖的引用和应用建议
- **常见错误** - 避坑指南
- **关键战术** - 可执行的战术表格
- **相关技能** - 互联的技能链接
