# 销售线索资质审核

## 何时使用此技能

当你需要：
- 评估潜在客户是否值得跟进
- 提高销售效率减少浪费
- 建立线索资质审核框架
- 优化销售管道质量

## 核心原则

### 1. 大多数销售问题是资质审核问题
如果你把时间花在错误的线索上，结果就是零。

### 2. 严格筛选节省资源
在不合格线索上投入的时间是完全浪费的。

### 3. 建立清晰的判断标准
需要有明确的框架来确定线索是否值得追求。

## 专家洞察

### Jen Abel
> "关键是资质审核。资质审核很重要，因为如果你把时间花在错误的线索上，那就等于零。"

**核心洞察**：这位专家明确指出大多数销售问题实际上是资质审核问题，并提供了确定线索是否值得追求的具体框架。

**如何应用**：
- 在投入时间之前严格评估每个线索
- 建立清晰的合格/不合格标准
- 愿意放弃不合格的线索
- 将资质审核视为销售成功的第一步

## 常见错误

- 追逐每一个看起来有兴趣的线索
- 没有明确的资质审核标准
- 在不合格线索上投入过多时间
- 用活动量而非质量来衡量工作
- 不愿意说"不"给不合格线索

## 关键战术

| 战术 | 说明 |
|------|------|
| **前置资质审核** | 在深入跟进前先评估线索质量 |
| **明确标准** | 建立清晰的合格/不合格判断框架 |
| **果断放弃** | 愿意放弃不符合标准的线索 |
| **质量优先** | 用管道质量而非数量来衡量工作 |
