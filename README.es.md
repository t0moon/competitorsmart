Herramienta de inteligencia competitiva asistida por IA para **Product Managers / PMM**.

## Inicio rápido

```bash
pip install -r requirements.txt
python main.py --competitors "CompetidorA,CompetidorB" --market "SaaS CRM"
```

- Salida de reportes: `outputs/agent/`
- Configura tu clave en `.env` (por ejemplo `ZHIPU_API_KEY_1`)
