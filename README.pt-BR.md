Kit de inteligencia competitiva com IA para **Product Managers / PMM**.

## Inicio rapido

```bash
pip install -r requirements.txt
python main.py --competitors "ConcorrenteA,ConcorrenteB" --market "SaaS CRM"
```

- Saida dos relatorios: `outputs/agent/`
- Configure sua chave no `.env` (por exemplo `ZHIPU_API_KEY_1`)
