**プロダクトマネージャー / PMM** 向けの AI 競合分析ツールです。

## クイックスタート

```bash
pip install -r requirements.txt
python main.py --competitors "競合A,競合B" --market "SaaS CRM"
```

- レポート出力先: `outputs/agent/`
- `.env` に API キー（例: `ZHIPU_API_KEY_1`）を設定してください
